# CLI コントラクト: cost-analyzer

## コマンドインターフェース

```
cost-analyzer [OPTIONS] QUERY
cost-analyzer serve [OPTIONS]
```

### `cost-analyzer QUERY` — 対話型クエリ

| 引数/オプション | 型 | 必須 | デフォルト | 説明 |
|---|---|---|---|---|
| `QUERY` | `str` | はい | — | 自然言語コストクエリ |
| `--format` | `str` | いいえ | `table` | 出力フォーマット: `table`, `json`, `csv` |
| `--lang` | `str` | いいえ | 自動検出 | 応答言語の強制: `ja`, `en` |

**終了コード**:
- `0` — 成功
- `1` — パースエラー（クエリが理解できない）
- `2` — 認証エラー
- `3` — API エラー（OCI が利用不可）
- `4` — 指定期間にデータなし

**出力（テーブルフォーマット）**:
```
┌─────────────────────┬────────────┬────────┐
│ サービス            │       コスト│     % │
├─────────────────────┼────────────┼────────┤
│ Compute             │   $1,234.56│  45.2% │
│ Object Storage      │     $678.90│  24.8% │
│ Networking          │     $456.78│  16.7% │
│ Database            │     $234.56│   8.6% │
│ Other               │     $128.34│   4.7% │
├─────────────────────┼────────────┼────────┤
│ 合計                │   $2,733.14│ 100.0% │
└─────────────────────┴────────────┴────────┘
期間: 2026-02-01 ~ 2026-02-28
```

**出力（比較）**:
```
┌─────────────────────┬────────────┬────────────┬───────────┬─────────┐
│ サービス            │       前期 │       当期 │      変化 │     %Δ │
├─────────────────────┼────────────┼────────────┼───────────┼─────────┤
│ Compute             │   $1,100.00│   $1,234.56│   +$134.56│  +12.2% │
│ Object Storage      │     $700.00│     $678.90│    -$21.10│   -3.0% │
│ ...                 │            │            │           │         │
├─────────────────────┼────────────┼────────────┼───────────┼─────────┤
│ 合計                │   $2,500.00│   $2,733.14│   +$233.14│   +9.3% │
└─────────────────────┴────────────┴────────────┴───────────┴─────────┘

サマリー: 合計コストが $233.14 (9.3%) 増加しました。Compute の増加額が
最大 (+$134.56)。Object Storage はわずかに減少 (-$21.10)。
```

**エラー出力 (stderr)**:
```
エラー: クエリを理解できませんでした。
以下の例を試してください:
  - "先月のサービス別コストを教えて"
  - "Show costs for February 2026"
  - "先月と今月を比較して"
```

### `cost-analyzer serve` — API サーバー

| オプション | 型 | 必須 | デフォルト | 説明 |
|---|---|---|---|---|
| `--host` | `str` | いいえ | `0.0.0.0` | バインドアドレス |
| `--port` | `int` | いいえ | `8080` | バインドポート |
