# UI コントラクト: Web UI フロントエンド

**ブランチ**: `002-web-ui` | **日付**: 2026-03-18

## ページ構成

### トップページ (`GET /`)

単一ページアプリケーション。クエリ入力と結果表示を1ページで完結する。

#### ヘッダー

- アプリケーション名: 「OCI コスト分析」/ "OCI Cost Analyzer"
- バックエンド接続状態インジケーター（緑=接続中、赤=切断）
- 言語切替トグル（JA / EN）

#### クエリ入力エリア

- テキスト入力欄（1行）
  - プレースホルダー: 「先月のサービス別コストを教えて」/ "Show costs for last month"
  - 最大文字数: 1000
- 送信ボタン
  - 入力欄が空のとき: 無効化（disabled）
  - 送信中: 無効化 + ローディングスピナー表示

## レスポンシブ対応

| 画面幅 | レイアウト |
|---|---|
| 768px 以上 (デスクトップ) | テーブルを全幅表示 |
| 360px 〜 767px (モバイル) | テーブルを水平スクロール可能なコンテナで表示 |

## 色分けルール

| 条件 | 色 | 用途 |
|---|---|---|
| 増加 (`absolute_change > 0`) | 赤系 (`#dc3545`) | 比較テーブルの変化額・変化率 |
| 減少 (`absolute_change < 0`) | 緑系 (`#28a745`) | 比較テーブルの変化額・変化率 |
| 変化なし (`absolute_change == 0`) | 中立（デフォルト色） | 比較テーブルの変化額・変化率 |
| エラー | 赤系 | エラーメッセージの背景/ボーダー |
| 確認 | 黄系 (`#ffc107`) | clarification メッセージの背景/ボーダー |
| バックエンド接続中 | 緑 | ヘッダーのステータスインジケーター |
| バックエンド切断 | 赤 | ヘッダーのステータスインジケーター |
